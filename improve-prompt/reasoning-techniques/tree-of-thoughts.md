# Tree of Thoughts (ToT)

Explore multiple distinct reasoning paths before selecting the best one.

**Use when:** the task has multiple plausible approaches and picking the wrong one early leads to failure (planning, creative writing with constraints, debugging).

**Gap it fixes:** premature commitment to a single reasoning path.

**Example:** `Consider three different approaches to this problem. For each, reason through it fully. Then select the best approach and explain why.`
