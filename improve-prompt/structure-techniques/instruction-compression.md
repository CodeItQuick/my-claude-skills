# Instruction compression

Rewrite system prompts and instructions to express the same intent in fewer tokens.

**Use when:** the system prompt or instruction block is verbose, repetitive, or contains prose that could be bullet points.

**Gap it fixes:** tokens wasted on phrasing instead of content, leaving less room for the actual input.

**Example:** Replace `Please make sure that you always respond in a friendly and professional tone and do not use jargon` with `Tone: friendly, professional, no jargon.`
