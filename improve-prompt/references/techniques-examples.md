# Example-Based Techniques — Index

Use these when the output format, tone, or style is hard to specify in words, or when the model needs a concrete reference for what "good" looks like.

Load the individual files from `../example-techniques/` based on what applies:

| Technique | When to reach for it | File |
|---|---|---|
| Zero-shot | Task is simple and unambiguous; examples would add noise | `../example-techniques/zero-shot.md` |
| One-shot | One example is sufficient; zero-shot produces inconsistent output | `../example-techniques/one-shot.md` |
| Few-shot | Output must match a format, tone, or style hard to describe in words | `../example-techniques/few-shot.md` |
| Negative examples | Positive examples alone don't prevent a known failure mode | `../example-techniques/negative-examples.md` |
