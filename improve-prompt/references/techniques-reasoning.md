# Reasoning Techniques — Index

Use these when the task involves multi-step thinking, logic, math, decisions, or tasks where intermediate steps determine correctness.

Load the individual files from `reasoning-techniques/` based on what applies:

| Technique | When to reach for it | File |
|---|---|---|
| Chain-of-thought (CoT) | Multi-step reasoning, math, logic | `../reasoning-techniques/chain-of-thought.md` |
| Zero-shot CoT | CoT needed but no examples available | `../reasoning-techniques/zero-shot-cot.md` |
| Tree of Thoughts (ToT) | Multiple plausible approaches; wrong early choice is costly | `../reasoning-techniques/tree-of-thoughts.md` |
| Self-consistency | Definite correct answer; single-path errors likely | `../reasoning-techniques/self-consistency.md` |
| Step-back prompting | Specific question hard to answer without general framework first | `../reasoning-techniques/step-back-prompting.md` |
| Least-to-most | Clear dependency structure; earlier answers feed later ones | `../reasoning-techniques/least-to-most.md` |
| Generated knowledge | Task requires domain context to be surfaced before reasoning | `../reasoning-techniques/generated-knowledge.md` |
